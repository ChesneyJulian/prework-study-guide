HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML Color Codes

CSS

  • CSS Resource
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Box-shadow adds shadow effects around an elements frame described by X and Y offsets relative to the element
  • External style sheets are preferred and must be linked in the head of the HTML file
  • Class attributes, expressed on HTML file, are styled on external style sheet and affect specified elements

Git

  • Git is a version control source
  • git add -A : commit all files in working branch
  • git commit -m "message" : commit files to repository with descriptive message
  • git pull origin main : syncs current branch with main branch
  • git push origin feature/example-html : pushes changes to github
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • Javascript Resource
  • J.S. adds interactivity to a website
  • J.S. is added to HTML at bottom of file before closing body tag
  • Control Flow is the order in which a computer executes code in a script
  • Javascript is executed from top to bottom
  • A variable is a named container that allows us to store data in our code
  • A function is a way of packaging functionality you want to reuse
  • Loops are used to efficiently execute code; for loops execute same code block over all items in an array
  • Arrays are single variables used to store a group of data
